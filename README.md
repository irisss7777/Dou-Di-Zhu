# Dou-Di-Zhu
<img width="776" height="557" alt="изображение" src="https://github.com/user-attachments/assets/a6420f36-c253-4f5e-aad4-afa80f475f7b" />

Dou-Di-Zhu - карточная онлайн игра. Взаимодействе с сервером осуществляется по WebSocket. Артихертурный подход выбран - CleanArchitecture + MVC(Подобное решение было принято на основании того, что бизнес логика вынеса на бекенд, на клиенте по сути отрабатывается input и выводиться View, оборачивать инпут в Application мне показалось нецелесообразным). Также в дизайне приложения я решил расширить модели и сделать их достаточно богатыми.

В рамках одного слоя взаимодействия происходят напрямую(между слоями по шине событий).

P.S: Взаимодействия между application и view я решил сделать напрямую через абстракцию-обертку(интерфейс)
